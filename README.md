# MemorySystem
```rust
流程开始 --> 记录实验者的姓名、性别和年龄信息 --> 生成6个随机数字 --> 显示数字给实验者 --> 实验者记忆数字 --> 实验者确认记忆完成 --> 生成下一个数字 --> 显示下一个数字给实验者 --> 实验者记忆数字 --> 实验者确认记忆完成 --> 生成下一个数字 --> 显示下一个数字给实验者 --> ... (重复此过程) --> 实验者表示没有记住某个数字 --> 倒退两个数字 --> 显示回退后的数字给实验者 --> 实验者记忆数字 --> 实验者确认记忆完成 --> 生成下一个数字 --> 显示下一个数字给实验者 --> ... (继续重复此过程) --> 实验者表示停止实验 --> 测试实验者记忆 --> 对比实验者复述的数字与正确的数字 --> 记录实验者记住的数字个数 --> 显示对比结果给实验者 --> 流程结束
```
